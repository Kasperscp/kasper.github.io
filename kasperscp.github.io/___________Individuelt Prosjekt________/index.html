Om

Denne nettsiden tar for seg en rekke med ulike spill. På hovedsiden kan du velge hvilke spill du ønsker å spille og dersom du ønsker å spille, kan du klikke på bildet av spillet du ønsker. Spillene er enkle og morsomme spill, som kan spilles når du ønsker. Spillene består av alt fra matematikk, til ping pong. Nettsiden ble opprettet den 20. Mars. 2023.
